# 📄 PDF ⇄ Image 변환 및 병합 도구

이 프로그램은 Python을 사용하여 PDF 파일을 이미지(PNG)로 분해하거나, 이미지들이 담긴 폴더를 다시 PDF로 병합해주는 유틸리티입니다.

## ✨ 주요 기능

1. **PDF 분해 (PDF ➡️ Images)**
* 현재 폴더의 PDF 파일을 탐색하여 목록을 보여줍니다.
* 원하는 PDF를 선택하면, 파일명과 동일한 폴더를 생성하고 각 페이지를 고해상도 PNG(`001.png`, `002.png`...)로 저장합니다.
* **선택 기능:** 원하는 파일 번호만 콕 집어서 변환할 수 있습니다. (예: `1 3` 또는 `all`)


2. **PDF 병합 (Images ➡️ PDF)**
* PNG 파일이 들어있는 폴더를 자동으로 감지하여 목록을 보여줍니다.
* 선택한 폴더 내의 이미지들을 순서대로 묶어 새로운 PDF로 만듭니다.
* **자동 네이밍:** 원본과 겹치지 않도록 `폴더명_converted.pdf` 이름으로 저장됩니다.



## 🛠️ 설치 및 준비

이 코드를 실행하기 위해서는 **Python 3**가 설치되어 있어야 하며, **PyMuPDF** 라이브러리가 필요합니다.

### 라이브러리 설치

터미널(또는 CMD)에서 아래 명령어를 입력하세요.

```bash
pip install pymupdf

```

## 🚀 사용 방법

1. 소스 코드가 있는 위치에 작업할 PDF 파일들을 둡니다.
2. 터미널에서 스크립트를 실행합니다.

```bash
python main.py
```

3. 메뉴에서 원하는 작업을 선택합니다.

```text
=== PDF 도구 모음 ===
1. PDF 분해 (선택한 PDF -> PNG)
2. PDF 병합 (선택한 폴더 -> PDF)
q. 종료
선택 > 

```

### 👉 선택 가이드

파일이나 폴더 목록이 나왔을 때, 아래와 같은 방식으로 선택할 수 있습니다.

* **단일 선택:** `1` (1번 항목만)
* **다중 선택:** `1 3 5` (띄어쓰기로 구분) 또는 `1, 2` (쉼표로 구분)
* **전체 선택:** `all` (목록의 모든 항목)

## 📂 파일 구조 및 동작 예시

작업을 수행하면 파일 구조는 아래와 같이 변경됩니다.

### 1. PDF 분해 시

```text
[Before]
📂 작업폴더/
 └─ 📄 report.pdf

[After]
📂 작업폴더/
 ├─ 📄 report.pdf
 └─ 📂 report/          <-- 자동 생성된 폴더
     ├─ 🖼️ 001.png
     ├─ 🖼️ 002.png
     └─ ...

```

### 2. PDF 병합 시

```text
[Before]
📂 작업폴더/
 └─ 📂 photos/
     ├─ 🖼️ img1.png
     └─ 🖼️ img2.png

[After]
📂 작업폴더/
 ├─ 📂 photos/
 └─ 📄 photos_converted.pdf  <-- 생성된 PDF 파일

```

## ⚠️ 주의 사항

* **이미지 순서:** 병합 시 이미지 파일 이름 순서(`sorted`)대로 페이지가 구성됩니다. (예: `001.png` -> `002.png`)
* **파일 제외:** 프로그램이 생성한 `_converted.pdf` 파일은 PDF 분해 목록에서 자동으로 제외되어 중복 작업을 방지합니다.
